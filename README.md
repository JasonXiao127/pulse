# Pulse

An extremely lightweight and barebones game launcher built with Flutter and Dart. `Pulse` gets out of your way and focuses on what matters: launching your games instantly without the bloat of traditional launchers. 

Steam relies on Chrome Embedded Framework, and the Epic Games Launcher somehow runs on Unreal Engine. Pulse uses Flutter instead, delivering significantly smaller binaries and a fraction of the RAM usage.

## Features

* **Barebones & Minimalist:** No heavy animations, store integrations, or background telemetry.
* **Featherweight Performance:** Built with Flutter for a tiny memory footprint and rapid startup times.
* **Clean UI:** A distraction-free interface designed to get you into your games faster.
